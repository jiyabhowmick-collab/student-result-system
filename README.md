# student-result-system
A simple Student Result Management System built using C programming concepts like functions, conditions, and loops.

This is a simple C program that calculates total marks, percentage, and grade of a student.

## Features
- Takes student name and 3 subject marks as input
- Calculates total and percentage
- Assigns grade based on percentage
